# Print method for summary.rmtfit objects

Produces a printed summary of the results for the restricted mean times
in favor of treatment

## Usage

``` r
# S3 method for class 'summary.rmtfit'
print(x, ...)
```

## Arguments

- x:

  An object returned by
  [`summary.rmtfit`](https://lmaowisc.github.io/rmt/reference/summary.rmtfit.md).

- ...:

  Further arguments passed to or from other methods

## Value

No return value, called for side effects.
